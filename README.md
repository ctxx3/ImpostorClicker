## Getting Started

To get a local copy up and running, follow these steps:

1. Clone the repository
```bash
git clone https://github.com/ctxx3/ImpostorClicker.git
```
2. Navigate to the project directory

```bash
cd ImpostorClicker
```
3. Install dependencies

```bash
flutter pub get
```

4. Run the app
```bash
flutter run
```
